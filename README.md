KRiPTON is a React + Express-based cryptocurrency trading simulator. It allows users to view real-time crypto data, buy and sell assets with virtual cash, and track their portfolio.

## Features
- Live cryptocurrency price data via RapidAPI.
- Buy and sell simulation with a starting balance of $10,000.
- Portfolio overview with holdings and remaining cash.
- News section for latest crypto updates.
- Modular React components and Express backend API.

## Tech Stack
- **Frontend:** React, Ant Design, Redux Toolkit, Chart.js
- **Backend:** Express, Node.js
- **APIs:** CoinRanking API (RapidAPI), NewsAPI
